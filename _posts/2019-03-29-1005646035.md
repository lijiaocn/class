## 课程介绍

软件环境： Jenkins & Pipeline + Kubernetes & Docker + Git + Maven + Harbor 
工作流程：手动/自动构建 -> Jenkins 调度 K8S API -＞动态生成 Jenkins Slave pod -＞ Slave pod 拉取 Git 代码／编译／打包镜像 -＞推送到镜像仓库 Harbor -＞ Slave 工作完成，Pod 自动销毁 -＞部署到测试或生产 Kubernetes平台 。
完全自动化，无序人工干预。
频发的交付带来更迅速的反馈，早发现问题早解决！
​
学完这门课程会获得什么？
这是一个项目实战。所以需要你具备Kubernetes，Jenkins，Git，Harbor等基础知识，才能更好的学习。
首先学习架构设计及目标，然后一步步部署这套架构，最后结合JAVA项目具体说明如何使用Jenkins实现企业CI（持续集成）/CD（持续部署/交付）及自动化。通过这个企业案例贯穿软件生命周期，学习实现思路及技巧，学完能独立打造一套健壮高效的CI/CD环境。
